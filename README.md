# EdurobV2-CAD
[Homepage](https://www.imsl.fh-dortmund.de/mobile-roboter/edurob/)

CAD files for Edurob V2

## Base
The base folder contains 3D printable file connecting
[PCB](https://github.com/IDiAL-IMSL/Edurob-PCB) and motors and provides a slot
for the USB power bank.

## Gripper
Contains files for a two finger gripper that uses a digital servo connected to
the EduRob PCB.

## Sensors
3D printable adapter to mount RPLidar A1M8 360° lidar sensor to the EduRob PCB.
